关于我们
合肥鹏锦自动化有限公司始建于2015年，是一家专业从事于非标自动化设备，工装夹具、检具研发制造销售于一体的生产厂商，公司多年来专注于解决机床与产品之间的连接，帮助客户分析产品特征，根据客户的设备，产能情况提供一流的解决方案，制造适合客户生产现场需求的工装夹具及非标设备。公司拥有一支高素质的从事产品研发、生产加工、总装调试、售后服务等专业人才队伍。公司具有完善的组织机构、具有完善的工艺、制造、装配、检测和质量管理制度。
